## How to get

1. [Download here](https://github.com/ModdingX/ModListCreator/releases)

OR

1. Clone this repository
2. Run `gradlew build`
3. Get file from path/build/libs/

## Detailed information

This tool works for both, Modrinth and CurseForge modpacks.
